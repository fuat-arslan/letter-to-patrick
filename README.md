# Canım Ablama — One-page letter

A self-contained, responsive one-page letter website. No external fonts, images, frameworks, or dependencies are required.

## Publish with GitHub Pages

1. Copy `index.html` into the root of your GitHub repository.
2. Commit and push it to the `main` branch.
3. Open the repository on GitHub.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Select the `main` branch and the `/ (root)` folder, then save.
7. GitHub will show the public URL after deployment.

The page can be edited directly inside `index.html`. The letter text begins under the `<section class="body-copy">` element.
